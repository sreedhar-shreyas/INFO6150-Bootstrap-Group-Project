List of Bootstrap components used:

1. Navbar: The entire navigation bar is enclosed within the <div class="navbar"> tag.
2. Badge: Used in the navbar for displaying notifications and a new post. Example: <span class="badge badge-pill badge-primary">New</span>
3. Icons: Bootstrap icons are used in the navbar, including bi-gear-fill and bi-person-circle.
4. Dropdown: Used in the navbar for the "My Profile" section. Example: <div class="navbar-item dropdown"> and <ul class="dropdown-menu" aria-labelledby="navbarDropdown">.
5. Container: Utilized to contain the main content within a fixed-width space. Example: <div class="container">.
6. Row and Column: Used within the container to organize the layout into rows and columns. Example: <div class="row">
