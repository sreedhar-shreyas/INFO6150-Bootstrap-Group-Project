Bootstrap component used
1. Form component- It is used in Login.html page to create a email address and password based login page.
2. Button component - It is used in Login.html page to create a login button. "btn  btn-primary" class is used here.
3. Pagination component - It is used in Login.html page to create Contact Us field.
4. Image component - It is used in Login.html page to add an image. "img-fluid img-thumbnail" class is used.
